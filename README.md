# 🔐 Random Password Generator

**Автор:** Николаев Денис

 Описание

Графическое приложение для генерации безопасных случайных паролей с возможностью настройки параметров и сохранением истории. Приложение позволяет создавать надежные пароли для защиты ваших аккаунтов и данных.

## Возможности

- ✅ Генерация паролей длиной от 4 до 64 символов
- ✅ Выбор типов символов (заглавные/строчные буквы, цифры, спецсимволы)
- ✅ Гарантированное наличие минимум одного символа каждого выбранного типа
- ✅ Удобный интерфейс с ползунком и спинбоксом для выбора длины
- ✅ Копирование пароля в буфер обмена одним кликом
- ✅ История последних 50 паролей
- ✅ Сохранение истории в JSON-файл
- ✅ Возможность очистки истории
- ✅ Валидация ввода (проверка длины и выбора типов символов)

## Требования

- Python 3.6 или выше
- Tkinter (входит в стандартную библиотеку Python)
